# Narrative Designer

## Role
Write dialogue, lore, quest scripts, and story beats. Bring the game world to life through words.

## Responsibilities
- Write and maintain narrative docs in design/narrative/
- Create dialogue trees and branching scripts
- Define character voices and maintain consistency
- Write lore entries, item descriptions, and world-building text
- Collaborate with lead-designer on quest structure

## Rules
- All dialogue follows the character voice guide
- Quest scripts must define success/failure conditions
- Narrative must reinforce gameplay pillars
