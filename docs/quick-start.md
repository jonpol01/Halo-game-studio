# Quick Start Guide

## Getting Started with Halo Game Studio

### Prerequisites
- Git initialized in the project root
- Game engine installed (Godot, Unity, or Unreal)

### First Steps

1. **Run `/start`** — The system detects your project state and guides you:
   - New project → brainstorm → GDD → sprint 0
   - Existing project → audit → gap analysis → integration

2. **Meet your team** — 12 AI agents organized in 3 tiers:
   - **Directors** (2): Creative Director, Technical Director
   - **Leads** (4): Design, Programming, Art, Production
   - **Specialists** (6): Gameplay, UI, Narrative, Audio, QA, DevOps

3. **Core workflow**:
   ```
   /sprint-plan → work → /code-review → /build-game → /playtest → /retrospective
   ```

### Key Commands
| Command | What it does |
|---------|-------------|
| `/start` | Project kickoff wizard |
| `/sprint-plan` | Plan next sprint |
| `/design-review` | Review a design doc |
| `/code-review` | Review code changes |
| `/build-game` | Produce a build |
| `/playtest` | Run a playtest session |
| `/bug-report` | File a bug |
| `/balance-check` | Analyze game balance |
| `/retrospective` | Sprint retrospective |
| `/team-combat` | Coordinate combat work |
| `/team-narrative` | Coordinate narrative work |
| `/team-ui` | Coordinate UI work |
