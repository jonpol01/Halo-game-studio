# Setup Requirements

## Minimum Requirements
- Git 2.30+
- A supported game engine:
  - Godot 4.x (recommended for rapid prototyping)
  - Unity 2022+ (recommended for 3D projects)
  - Unreal 5.x (recommended for AAA-quality visuals)

## Configuration
1. Clone this repository
2. Run `/start` to initialize the project
4. Configure your game engine project in `src/`

## Directory Permissions
- `src/core/` — Protected. Requires technical-director approval to modify.
- Config files — Protected. Requires Director approval.
- `prototypes/` — Open. Anyone can write here. Code is never promoted to src/ without review.

## Recommended Workflow
1. Sprint planning (weekly or biweekly)
2. Daily: implement, review, test
3. Sprint end: build, playtest, retrospective
4. Milestone: release candidate, full QA pass
