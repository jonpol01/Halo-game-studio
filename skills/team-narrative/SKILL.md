# /team-narrative — Narrative Team Orchestration

## Trigger
Run when building a quest, story beat, or narrative-heavy feature.

## Team
- narrative-designer → story, dialogue, quest scripts
- lead-designer → quest structure, rewards, pacing
- gameplay-programmer → quest system implementation
- audio-designer → voice direction, ambient audio
- ui-developer → dialogue UI, quest log

## Workflow
1. narrative-designer writes the story outline
2. lead-designer defines quest structure and rewards
3. narrative-designer writes full dialogue and scripts
4. gameplay-programmer implements quest logic
5. ui-developer builds dialogue and quest UI
6. audio-designer adds voice and ambient audio
7. /playtest the narrative content
8. Iterate on pacing and clarity
