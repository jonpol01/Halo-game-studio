# /team-ui — UI Team Orchestration

## Trigger
Run when building a major UI feature (menu system, HUD overhaul, inventory).

## Team
- ui-developer → implementation in src/ui/
- lead-artist → visual design, style guide compliance
- lead-designer → UX flow, information hierarchy
- audio-designer → UI sound effects
- qa-tester → UI test plan (responsiveness, edge cases)

## Workflow
1. lead-designer defines the UX flow and wireframes
2. lead-artist creates visual mockups
3. ui-developer implements the UI
4. audio-designer adds UI sounds
5. qa-tester tests across screen sizes and input methods
6. /design-review for final approval
7. Iterate based on feedback
